# Unit Testing Guide

## 1. Overview
This guide explains how to verify the functionality of the Enhanced Mock LLM Service. The service includes a suite of 15 automated unit tests that verify response logic, variability, and latency.

## 2. Running the Tests
The project is set up with Maven and JUnit 5.

### Option A: Command Line (Recommended)
Run the full test suite from the terminal:
```bash
mvn test
